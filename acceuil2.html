Les commentaires sont des instructions qui n’auront aucun effet sur le document. Ils servent à ajouter des instructions ou explications sur certaines parties du document. Le concept de commentaire est présent dans toutes les languages de programmation. Il s’agit d’une bonne pratique qui permet d’expliquer une portion de code à ses collaborateurs. Pour écrire un commentaire en HTML on utilise la syntaxe suivante . Tout le texte entre sera considéré comme un commentaire et donc, ne s’affichera pas sur la page Web. Les commentaires peuvent aussi être utilisés pour désactiver une partie du code.
